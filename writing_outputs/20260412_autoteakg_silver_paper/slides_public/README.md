# AutoTeaKG-Silver Public Talk

Audience: general / non-specialist audience.

Suggested duration: 12-15 minutes.

Files:

- `AutoTeaKG_Silver_public_talk.pptx`: PowerPoint deck with 13 slides and speaker notes.
- `AutoTeaKG_Silver_public_talk_script.md`: full Chinese speaker script, organized slide by slide.

Design intent:

- Use a concrete public-facing narrative: tea claims are common, but evidence is fragmented.
- Avoid overclaiming health effects.
- Explain AutoTeaKG-Silver as an evidence map rather than a recommendation engine.
- Keep slides visually simple and readable.

Note:

- `pptx-posters` was not used because this is a slide deck, not a poster.
- PDF export was not generated automatically because LibreOffice/soffice is not available in the current environment.
