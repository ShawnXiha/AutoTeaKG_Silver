## Main Lessons From Adjudication

1. `effect_direction` must be used carefully for reviews and meta-analyses. Some records should be `mixed` rather than `positive`.
2. The current schema is sufficient for phase 1 and phase 2.
3. For phase 3 microbiome KG work, taxon names should remain conservative until verified from abstracts/full text.
4. Review-derived records are useful for seeding and mapping, but downstream quantitative summaries should distinguish them from primary-study records.
